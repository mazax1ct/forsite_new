СЕРВЕР NVIDIA RTX

Высокопроизводительные визуальные вычисления
в дата-центре

Гибкое референсное решение NVIDIA RTX™ Server обеспечивает ускорение RTX на любом устройстве. Сервер объединяет в себе передовые графические карты NVIDIA® 6000 и 8000 с ПО NVIDIA Virtual GPU (vGPU), обеспечивая исключительную вычислительную мощность.

Благодаря ряду сертифицированных решений, от виртуализации до совместного проектирования на NVIDIA Omniverse™, RTX Server позволяет работать продуктивно из любой точки, требуя меньше затрат, площади и электроэнергии по сравнению с традиционными решениями на базе CPU.

Ускорение различных нагрузок на ПК и в дата-центре


Рендеринг

Мощность GPU ускоряет финальный рендеринг или повышает производительность локальной рабочей станции. Так как сервер NVIDIA RTX оснащен программным обеспечением с ускорением RTX, графические художники могут выполнять офлайн-рендеринг и создавать финальные изображения с трассировкой лучей быстрее, чем с традиционными рендер-фермами на базе CPU. Графические художники могут обеспечить интерактивную визуализацию в области просмотра приложения, подключаясь к одному или нескольким серверам RTX, чтобы повысить производительность.



Совместная работа над проектами

Путь от идеи к воплощению — это многоэтапный процесс, который требует тесного сотрудничества между дизайнерами и разработчиками. С ведущей платформой для совместной работы NVIDIA Omniverse дизайнеры могут упростить работу с графикой в реальном времени. Omniverse на базе RTX Server с ПО NVIDIA vGPU упрощает работу с 2D- и 3D-графикой для дизайнеров во многих отраслях. Созданная с учетом современных отраслевых стандартов для работы с графикой платформа поддерживает постоянное соединение между приложениями и рендерером NVIDIA Omniverse RTX.



Производительные виртуальные рабочие станции

RTX Server для виртуальных рабочих станций позволяет работать с графикой и видео в любом месте, обеспечивая возможности платформы Quadro RTX в виртуализированной среде. Один сервер NVIDIA RTX, оснащенный ПО Quadro® Virtual Data Center Workstation (Quadro vDWS), предоставляет доступ к нескольким простым в управлении виртуальным рабочим станциям или к комплексу виртуальных рабочих станций и узлов для рендеринга. Решение также позволяет виртуализировать нагрузки обработки данных, чтобы выполнять самые ресурсоемкие проекты по аналитике и вычислениям удаленно с помощью ПО NVIDIA Virtual Compute Server (vCS) .



Системы автоматизированного проектирования (САПР)

Инженерное моделирование требует большого объема вычислений. Сервер RTX Server и ПО Quadro vDWS обеспечивают более быстрое создание, тестирование и итерации проектов благодаря высокой вычислительной мощности для проектирования и вычислений, что позволяет инженерам работать в САПР из любого места. Параллельная обработка на GPU ускоряет коды моделирования на 32-битных системах, а тензорные и RT ядра в архитектуре NVIDIA Turing™ помогают приложениям с поддержкой RTX быстрее получать более точные результаты вычислений и рендеринга. Возможность использовать несколько графических процессоров NVIDIA с помощью Quadro vDWS обеспечивает более быстрое, плавное и безопасное моделирование благодаря обработке высокоточных моделей в среде VDI.



AR и VR на периферийных устройствах

Обеспечение рендеринга на GPU и передачи графики по сети — это сложный процесс, который требует опыта в области оркестрации облачных систем, виртуализации, работы сетей 5G и других сферах. Ускорить передачу контента по сети позволит NVIDIA RTX Server для передачи приложений расширенной реальности (XR) по сети мультисервисного доступа 5G. Это комплексное программно-аппаратное решение NVIDIA на базе сертифицированных систем RTX Server. Решение объединяет в себе GPU Quadro RTX, ПО Quadro vDWS и SDK NVIDIA CloudXR™, что позволяет разработчикам масштабировать XR-приложения и передавать контент на портативные устройства по сети 5G.



Масштабируемые решения для визуализации

Создавайте масштабные динамичные визуализации при помощи NVIDIA RTX Server, который подходит для таких задач и позволяет легко развернуть большие среды и управлять ими из дата-центра. С RTX Server вы можете создавать максимально иммерсивные визуальные проекты, будь то разработка нового аттракциона для парка отдыха или комнаты виртуальной реальности (CAVE).

NVIDIA Omniverse™ Platform

NVIDIA Omniverse is a powerful, multi-GPU, real-time simulation and collaboration platform for 3D production pipelines based on Pixar's Universal Scene Description and NVIDIA RTX™.

Omniverse aims for universal interoperability across different applications and 3D ecosystem vendors. It provides efficient real-time scene updates and is based on open-standards and protocols. The Omniverse Platform is designed to act as a hub, enabling new capabilities to be exposed as micro-services to any connected clients and applications.
