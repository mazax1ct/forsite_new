Видеостена для службы безопасности

Объект г. Санкт - Петербург,
ул. Сименса, д.1

Категории Проект
Видеостены

Дата Q1 2016 года

Сайт компании ww.netbynet.ru

Задача:

Монтаж видеостены состоящей из 6 тонкошовных профессиональных панелей 46", вывод нескольких физических источников одно из которых устройство для беспроводной презентации

Реализация:

Изначальный конструктив, изготовленный предыдущим инсталлятором не выдержал вес панелей и согнулся, повредив при этом панели. Наша задача состояла в том чтобы сделать жесткий конструктив с полной зашивкой, но в то же время с возможностью перемещения по залу.

К тому же обеспечить возможность удаленного запуска различного рода рекламных видео (прямая Digital Signage - задача) с возможностью переключения на внешнее, принесенное устройство.

К сожалению, для уменьшения веса конструкции пришлось отказаться от выдвижных модулей для видеопанелей. Это немного усложнило задачу нащим техническим специалистам, но они с ней справились. В качестве видеконтроллера было выбрапно компактное решение FORSITE NVS-100 c выходом 4K, для получения максимального разрешения по цепочке. Управление DS контентом доверили надежному ПО PADS4.
